# Global Marine Conservation Map

An interactive global mapping tool developed for policy research at the University of Oxford, visualising marine protected area regulations alongside coral reef thermal stress, mangrove extent, and live biodiversity data.

## Research Questions

1. Which marine protected areas are failing under climate stress?
2. Where are high-biodiversity areas currently unprotected?
